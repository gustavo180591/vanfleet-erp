# vanfleet-erp

ERP-CRM ligero para gestión de alquiler de furgonetas.

## Objetivo

- Gestión de clientes
- Gestión de flota (furgonetas)
- Contratos de alquiler
- Facturación + Verifactu
- Panel de beneficios por vehículo y global

## Stack (propuesto)

- Backend/Frontend: Laravel 11 (monolítico)
- Base de datos: MySQL/PostgreSQL
- UI: Blade + Tailwind CSS
