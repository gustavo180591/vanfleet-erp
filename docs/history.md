# Especificación de Requisitos - Sistema de Gestión para Alquiler de Furgonetas

## Visión General
Sistema ERP-CRM ligero para la gestión integral de una empresa de alquiler de furgonetas, con foco en simplicidad y usabilidad.

## Objetivos Principales
- Digitalizar y centralizar la gestión de clientes, vehículos y contratos
- Automatizar procesos de facturación y seguimiento
- Proporcionar visibilidad en tiempo real del estado de la flota
- Generar reportes básicos de rentabilidad

## Módulos Principales

### 1. Gestión de Clientes
- **Registro de Clientes**
  - Datos personales y de contacto
  - Documentación (DNI, carnet de conducir, etc.)
  - Historial de alquileres
- **Perfil del Cliente**
  - Historial de alquileres
  - Incidencias reportadas
  - Estado de pagos

### 2. Gestión de Flota
- **Ficha Técnica por Vehículo**
  - Datos del vehículo (matrícula, modelo, año)
  - Estado actual (disponible/alquilado/mantenimiento)
  - Historial de mantenimiento
- **Disponibilidad**
  - Calendario de disponibilidad
  - Alertas de mantenimiento programado

### 3. Gestión de Contratos
- **Generación de Contratos**
  - Plantillas personalizables
  - Cálculo automático de precios
  - Firma digital
- **Seguimiento**
  - Vencimientos
  - Renovaciones
  - Documentación asociada

### 4. Facturación y Reportes
- **Facturación Automática**
  - Cálculo de tarifas
  - Generación de facturas
  - Integración con Verifactu
- **Reportes**
  - Ingresos por vehículo
  - Tasa de ocupación
  - Rentabilidad

## Requisitos Técnicos
- Plataforma web responsive
- Base de datos segura con copias de seguridad
- Exportación a formatos estándar (PDF, CSV, Excel)
- Autenticación de usuarios con roles

## Entregables
1. Aplicación web funcional
2. Documentación técnica
3. Manual de usuario
4. Código fuente

## Restricciones
- Presupuesto: €250-750 EUR
- Tiempo de entrega: 2 semanas
- Debe ser fácil de mantener y escalar
